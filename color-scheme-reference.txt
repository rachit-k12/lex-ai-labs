DESIGN SYSTEM SUMMARY
═══════════════════════════════════════════════════════════════════════

PRIMARY PALETTE (Most Used Colors):
-----------------------------------
1. slate-900 (#0f172a) - Headlines, dark buttons, primary text
2. slate-500 (#64748b) - Body text, descriptions
3. brand-600 (#2563eb) - Primary brand color, CTAs, accents
4. white (#ffffff) - Backgrounds, light text
5. surface-50 (#f8fafc) - Alternating section backgrounds

SUPPORTING COLORS:
------------------
• slate-50 to slate-800 - Various UI elements
• brand-400 to brand-500 - Interactive states
• emerald-500/600 - Success/free elements
• amber-400 - Star ratings
• #0077b5 - LinkedIn brand

COLOR STRATEGY:
---------------
1. Clean & Professional - Heavy use of grays and whites
2. Blue Accent - Brand blue (#2563eb) used sparingly for impact
3. High Contrast - Dark text on light backgrounds for readability
4. Subtle Variations - Multiple slate shades for depth without color
5. Strategic Color - Color only where it adds meaning (CTAs, ratings, brands)

ACCESSIBILITY & CONTRAST RATIOS
═══════════════════════════════════════════════════════════════════════

PRIMARY TEXT COMBINATIONS (WCAG AA Compliant):
-----------------------------------------------
✓ slate-900 on white - 16.1:1 (AAA) - Headlines
✓ slate-600 on white - 8.6:1 (AAA) - Secondary text  
✓ slate-500 on white - 7.5:1 (AAA) - Body text
✓ white on slate-900 - 16.1:1 (AAA) - Dark sections
✓ white on brand-600 - 5.9:1 (AA) - CTA buttons

CAUTION COMBINATIONS:
---------------------
⚠ slate-400 on white - 4.5:1 (AA) - Minimum for small text
⚠ brand-500 on white - 3.5:1 - Use for large text only

═══════════════════════════════════════════════════════════════════════
MOBILE vs DESKTOP COLOR CONSIDERATIONS
═══════════════════════════════════════════════════════════════════════

MOBILE-FIRST APPROACH:
----------------------
• Higher contrast maintained at all sizes
• Larger touch targets use consistent colors
• Dark text on light backgrounds (no thin gray on white)
• CTAs maintain brand-600 blue for visibility
• No color-only indicators (combined with icons/text)

RESPONSIVE COLOR BEHAVIOR:
--------------------------
• Colors remain consistent across breakpoints
• No color changes on resize
• Shadow intensity may vary for performance
• Hover states disabled on touch devices
